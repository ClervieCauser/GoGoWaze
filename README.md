# GoGoWaze

TD2 - Accidents

# Introduction
Le but du projet est de créer une application mobile afin de signaler en temps réel un accident de la route. En créant ces accidents on peut leur mettre une gravité et une photo permettant à un utilisateur de recevoir des notifications.
L'applicatiion est adapté à tout type d'utilisateurs sur tablette ou téléphone.

# Axes :
- Clervie CAUSER : Gérer (ajouter/supprimer/modifier) des informaitons dynamiquement à partir d'un Web Service en ligne.
- Louis DEDIEU : Créer une notification utile avec temps d'affichage limitée. + Visualiser une notification avec des images en entête.
- Amandiinne MARTIN : Intégrer OpenStreetMap dans l'application, pour indiquer de lieux.
- Léo MENDES : Afficher des informations dynamiquement à partir d'un Web Service en ligne. + Implémenter une idée qui utilise les capteurs du dispositif du type photo.
- Julien ORIOL :  Faire une application adaptable pour un affichage différent pour tablette et smartphone. + Gérer l'orientation paysage/portait dans les différentes vues du téléphone, principalement lorsqu’elle apporte un plus à l’application.

# MVC + Pattern Factory :
Léo s'est occupé du MVC dans sa partie. Et avec l'aide d'Amandine ils ont ajouté une Simple Factory toujours dans la même partie.

# Les Fragments :
Nous avons des fragments un peu partout. Par exemple, pour signaler un accident on clique sur le menu qui ouvre un fragment où l'on peut signaler et donc ajouter un accident sur la carte avec des Web service.

# Permissions :
Nous avons utilisé les permissions pour la photo de profil et l'ajout d'une photo pour la signalisation.
